# LinkedListExample.java

A Java program showcasing the fundamentals of **Linked Lists** — one of the core data structures in computer science.  
This file demonstrates how nodes are connected dynamically and how common operations are performed step‑by‑step.

## Features
- Implementation of a **singly linked list**  
- Node creation and linking  
- Insertion at head, tail, and specific positions  
- Deletion of nodes  
- Traversal and printing of list elements  
- Clear, beginner‑friendly comments for learning  

## Usage
Compile and run:
```bash
javac LinkedListExample.java
java LinkedListExample
